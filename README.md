# Проект: Сложно сосредоточиться (No Focus)

## Ссылка на проект

https://github.com/Yulia-Pospelova/slozhno-sosredotochitsya-fd

Адаптивный лендинг с динамическим переключением цветовых схем (Dark/Light/Auto).

## Технологии в проекте

- **HTML5**: Семантическая разметка.
- **CSS3**:
  - Flexbox и Grid Layout для построения сетки.
  - CSS Variables (Custom Properties) для темизации.
  - `clamp()` для резиновой типографики.
  - `color-mix` для работы с прозрачностью.
  - Медиа-запросы для адаптивности и системных тем (`prefers-color-scheme`).
- **JavaScript**: Логика переключения классов темы на корневом элементе.

## Функционал

1. **Тема "Неон"**: Темная тема, установленная по умолчанию.
2. **Тема "День"**: Светлая тема с улучшенной контрастностью для чтения.
3. **Режим "Авто"**: Подстраивается под системные настройки пользователя.
4. **Адаптивность**: Сайт корректно отображается на устройствах от 320px до 1024px и выше.

## Как запустить

Просто откройте файл `index.html` в вашем браузере.
